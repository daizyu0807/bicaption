# 術語表 Terminology Glossary

| Term | Definition |
|---|---|
| STT | Speech-to-Text，將語音轉為文字 |
| Partial Transcript | 尚未穩定、可能被後續修正的字幕片段 |
| Final Transcript | 已穩定、不再回寫的字幕片段 |
| Overlay | 疊加在螢幕上的字幕視窗 |
| Provider | 實際執行 STT 或翻譯的引擎實作 |
| BlackHole | macOS 虛擬音訊裝置，用於 app 間音訊路由 |
