# 商業模型 Business Model

## 目標市場 Target Market
- 初期: 個人知識工作者、跨語言會議參與者
- 後續: 研究、教育、遠端協作場景

## 價值主張 Value Proposition
- 不依賴會議平台方案限制
- 即插即用地為任何音訊來源提供雙字幕
- 保留本機優先與隱私優先路線

## 收入來源 Revenue Streams
- MVP 階段: 無
- 後續可選:
  - 一次性桌面授權
  - 進階 provider 設定與專業版功能
  - 團隊版部署 [NEEDS_CONFIRMATION]

## 成本結構 Cost Structure
- 開發時間
- STT / 翻譯 API 費用 [若採雲端]
- 封裝與發行維護成本
