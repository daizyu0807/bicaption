# BiCaption Icon Concepts

Two rounds of icon directions for BiCaption.

## Minimal set

- `minimal-wave-captions.svg`
  - Simplest "subtitle + wave" read
  - Best balance between product meaning and simplicity
- `minimal-subtitle-pulse.svg`
  - Darker, more technical
  - Subtitle bars with a pulse-like center stroke
- `minimal-wave-window.svg`
  - Most literal window metaphor
  - Closest to an app overlay abstraction

## First-pass set

- `speech-orb.svg`
  - Dictation-first
  - Warm orb with waveform cut
  - Strong match for the hold-to-talk recording orb
- `dual-caption-tiles.svg`
  - Product-first
  - Two caption layers plus microphone mark
  - Clearest "bilingual subtitle + dictation" metaphor
- `b-monogram-wave.svg`
  - Brand-first
  - Stylized `B` with embedded voice wave
  - Strongest long-term logo direction

Open the SVG files directly in Finder, a browser, or design tools for review.
